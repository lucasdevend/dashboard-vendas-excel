# Dashboard de Vendas no Excel

## Objetivo
Criar um dashboard de vendas no Excel com base em uma base de dados fornecida pelo curso, transformando dados brutos em informações visuais claras para apoiar a análise e a tomada de decisão.

## Dados Utilizados
Foi utilizada a base de dados `base.xlsx`, que contém informações relacionadas às vendas.

## Dashboard
O dashboard foi desenvolvido no Microsoft Excel utilizando tabelas dinâmicas e gráficos, permitindo uma visualização simples e organizada dos dados de vendas.

## Como Reproduzir
1. Abrir o arquivo `dashboard.xlsx` no Microsoft Excel.
2. Caso deseje recriar o dashboard, utilizar a base `base.xlsx` e criar tabelas dinâmicas e gráficos a partir dos dados.

## Ferramentas Utilizadas
- Microsoft Excel
- GitHub
